# Seminar Registration App

A modern Android application for seminar registration built using **Kotlin** and **XML UI (ConstraintLayout)**.
This app features a clean and interactive UI with smooth animations and user-friendly design.

---

## ✨ Features

* 🔐 Login validation (Email & Password)
* 🏠 Modern Dashboard UI with:

  * Curved Header Design
  * Personalized Greeting
  * Highlight Seminar Card
  * Upcoming Seminar List
* 📝 Seminar Registration Form
* ✅ Result Page (Registration Summary)
* 🎬 Smooth UI Animations (Slide & Fade)
* 🎨 Clean & Modern UI Design

---

## 📸 Preview

> UI inspired by modern mobile app design

* Dashboard with gradient header
* Interactive cards & buttons
* Minimalist and responsive layout

---

## 🛠️ Tech Stack

* **Language**: Kotlin
* **UI**: XML (ConstraintLayout, LinearLayout)
* **IDE**: Android Studio
* **Design Style**: Material + Custom UI
